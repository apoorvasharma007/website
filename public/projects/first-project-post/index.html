Minimal Personal Site with Hugo

26 Feb 2026 Projects

This project is a deliberately minimal personal site: no JavaScript bundle, no theme dependency, and a small amount of CSS that I can understand at a glance.

The goal is to have a home for long-form writing on cloud engineering, notes on fitness, and concise write-ups of projects I care about—without taking on a maintenance burden.
